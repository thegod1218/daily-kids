# Daily Kids

아이 생활 데이터 기반 가족 플랫폼 MVP입니다.

## 현재 구현 범위
- 스플래시
- 시작 화면
- 이메일 회원가입
- 로그인
- 메일 인증 안내
- 부모 프로필 설정
- 아이 등록
- 첫 홈 화면 목업

## 실행 방법
```bash
npm install
npm run dev
```

## Vercel 배포
1. GitHub에 이 프로젝트 업로드
2. Vercel에서 New Project 선택
3. Framework Preset: Vite
4. Build Command: npm run build
5. Output Directory: dist
